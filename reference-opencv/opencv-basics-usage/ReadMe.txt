Description about .py files

camera_in.py: 웹캠으로 찍는 거 실시간으로 읽기

camera_out.py: 웹캠에서 찍은 거 (쓰기) 저장

video_clip_in.py: video clip 파일 읽기

video_clip_out.py: video clip 파일 (쓰기) 저장

mask_ROI, mask_ROI2.py: Region of Interest - cv2.copyTo() 예시

keyboard.py: 키보드 제어 예시

mouse.py: 마우스 event, flags 처리 예시

trackbar.py: 트랙바 인터페이스 사용 예시 - cv2.createTrackbar()

image_info.py: np.ndarray, image 인덱싱

draw_circle_rec_text.py: circle, rectangle, text, polyline 예시

tick.py: 시간 측정 예시 - cv2.TickMeter()

video_effect.py : 동영상 합성 예시 (for 문 사용)
# cv2.VideoCapture.read() 는 frame을 yield!
